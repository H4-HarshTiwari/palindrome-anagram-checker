# Palindrome & Anagram Checker

A console-based Java tool to check whether a string is a palindrome (ignoring case and non-alphanumeric characters) and whether two strings are anagrams of each other.

## Features
- Palindrome check using two-pointer technique, skipping spaces/punctuation and ignoring case
- Anagram check using a character-frequency array (no HashMap)
- Simple looping console menu — check multiple strings without restarting

## Concepts Used
- Two-pointer technique (O(n))
- Character-to-index mapping using ASCII arithmetic (`ch - 'a'`)
- Array-based frequency counting
